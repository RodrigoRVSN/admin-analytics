### Getting Started

- Set up the [server repository](https://github.com/biantris/admin-analytics-api) (note: it needs to be running together with the frontend)
- Clone this repo

```sh
# install dependencies
yarn
# or
yarn install

# start project
yarn dev

# see on localhost link
http://localhost:3000
```
